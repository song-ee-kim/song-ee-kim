# Profile

I am an Operations Analyst / Data Analyst with experience in
operations data analysis, SOP design, and process improvement
across public and large-scale organizational environments.

My work focuses on defining operational metrics, identifying
error patterns, and validating improvement initiatives through
structured experimentation and data-driven analysis.

I am particularly interested in operational domains where
accuracy, standardization, and scalability directly impact
business outcomes.

---

## Areas of Focus

- Operational data analysis and accuracy metrics
- SOP design and workflow standardization
- Error pattern analysis and root cause identification
- A/B testing and impact validation
- Translating analysis into actionable operational improvements

---

## Featured Project

### Catalog Ops Error Reduction via A/B Testing

- Analyzed catalog labeling errors using simulated operation logs
- Identified higher error rates among new operators
- Designed SOP improvements and validated impact via A/B testing
- Achieved a 27% relative reduction in error rate
  (from 15.4% to 11.3%)
